# Dart example 3rd party plugin

Example of a 3rd party plugin written in dart and compiled to js.

## How to use

1. Write your code in main.dart. You may also create additional folders and .dart files and import
   them in main.dart. Just make sure that at the end all functions are implemented and can be
   accessed from main.dart
2. Don't edit wrapper.dart and bridge-functions.js
3. Run compile.sh to compile into a ready plugin.zip
